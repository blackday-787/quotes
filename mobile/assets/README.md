# Assets Placeholder

Expo will generate default assets when you first run the app.

To customize your app icons:
1. Create icon.png (1024x1024)
2. Create splash.png (recommended 2048x2048)
3. Create adaptive-icon.png (1024x1024) for Android
4. Create notification-icon.png (96x96) for notifications

For now, Expo will use default assets.
