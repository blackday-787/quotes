## Step 2: Add Test Quotes (1 minute)

- "Test quote 1"
- "Test quote 2"
- "Test quote 3"

## Step 3: Start Mobile App (2 minutes)

```bash
# New terminal
cd mobile
npm install
npm start
```

1. Scan QR code with iPhone
2. Grant notification permissions
3. Tap "Connect to Server"
4. Tap "Send Test Quote"
